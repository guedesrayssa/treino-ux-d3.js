# Ice Cream Maker

- **o que é**  
  desenhozinho de sorvete feito com d3.js e svg, com casquinha, 3 bolas, tubinho e cereja.

- **como usar**  
  1. abra o arquivo `index.html` no navegador.  
  2. clique em cada **bola de sorvete** para trocar as cores dos sabores.  
  3. clique na **casquinha** para mudar o tom do cone.  
  4. clique no **tubinho** para mudar o tom de chocolate.
